# 🏢 Wicket-Company v3.2

_A Butter-Smooth Modular Company Ops Assistant — by Adam Clark (2026)_

---

## Features
- Persistent, modular skills (website/SEO, data entry, marketing, notes)
- Analytics dashboard, CSV export, "proactive" onboarding and stats
- File attachment tracker, user roles, secure memory (auto-saves to JSON)
- AI copilot (OpenAI GPT if key present, fallback otherwise)
- Recruiter-focused Easter egg commands built-in
- Ready for API/cloud integration and rapid scale-up

---

## How to Run

1. *(Optional for AI)* `pip install openai` & export your OpenAI key  
    `export OPENAI_API_KEY=sk-...`

2. In Terminal:
    ```sh
    python3 wicket_company.py
    ```

3. Select a menu option or type `hireme`/`recruitme/67890` for a secret message!
